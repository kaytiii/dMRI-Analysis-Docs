Diffusional Kurtosis Estimator (DKE) User's Guide

Overview

Diffusional Kurtosis Estimator (DKE) is a program for post-processing diffusional kurtosis imaging (DKI) datasets. DKE consists of a suite of command-line programs along with a grahpical user interface (GUI). The command-line programs run on 32- and 64-bit Windows, as well as on 64-bit Linux platforms. The GUI frontend runs on 64-bit Windows platforms. DKE supports DICOM and NIfTI formats.

If you use DKE in a publication, please cite the following paper in addition to standard DKI references:

Tabesh A, Jensen JH, Ardekani BA, and Helpern JA. Estimation of tensors and tensor-derived measures in diffusional kurtosis imaging. Mag Reson Med. 2011 Mar;65(3):823-36. http://www.ncbi.nlm.nih.gov/pubmed/21337412
